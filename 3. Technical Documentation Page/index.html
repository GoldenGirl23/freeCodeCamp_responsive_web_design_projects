1 Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its use of significant whitespace. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python is dynamically typed and garbage-collected. It is an ideal language for scripting and rapid application development in many areas on most platforms. The comprehensive standard library and its vast ecosystem of third-party packages make it a powerful tool for developers.

2 Data Types and Variables

In Python, variables are used to store data, and they do not require explicit declaration to reserve memory space. The declaration happens automatically when a value is assigned to a variable. Python supports several data types including:

  • Numbers: int, float, complex.
  • Strings: A sequence of characters enclosed in quotes.
  • Lists: Ordered, mutable collections of elements.
  • Tuples: Ordered, immutable collections of elements.
  • Dictionaries: Unordered collections of key-value pairs.
  • Sets: Unordered collections of unique elements.

Example:


        x = 5          # int
        y = 3.14       # float
        name = "John"  # string
        numbers = [1, 2, 3]  # list
        point = (1, 2)  # tuple
        user = {"name": "Alice", "age": 25}  # dictionary
        unique_numbers = {1, 2, 3}  # set

3 Control Flow

Control flow statements are used to control the execution of the program based on certain conditions. Python includes several control flow statements:

If-Else Statements

The if statement is used for conditional execution. It executes a block of code if a specified condition is true. Optionally, an else statement can be used to execute a block of code if the condition is false.

Example:


        age = 18
        if age >= 18:
            print("You are an adult.")
        else:
            print("You are a minor.")

Loops

Python provides two types of loops: for and while.

  • For Loop: Iterates over a sequence (such as a list, tuple, dictionary, set, or string).
  • While Loop: Repeats as long as a specified condition is true.

Example:


        # For Loop
        for i in range(5):
          print(i)

        # While Loop
          count = 0
          while count < 5:
            print(count)
            count += 1

4 Functions

Functions are blocks of reusable code that perform a specific task. They help to break our program into smaller and modular chunks. As our program grows larger and larger, functions make it more organized and manageable.

A function is defined using the def keyword, followed by the function name and parentheses containing any parameters. Functions can be passed arguments and can also return values using the return statement.

Example:


        # Defining a function
        def greet(name):
          print(f"Hello, {name}!")

        # Calling the function
        greet("Alice")

        # Returning values from a function
        def add(a, b):
          return a + b

        result = add(3, 4)
        print(result)  # Output: 7

5 Modules and Packages

Modules and packages in Python help to organize and structure your code effectively in a modular format. A module is a single file containing Python code, whereas a package is a collection of modules that are organized in a directory hierarchy. Modules and packages can be imported and used in another Python program.

Example:


        import math_module

        result = math_module.add(5, 3)
        print(result)  # Output: 8

Python is a versatile and powerful programming language that can be used for a wide range of applications. Its simplicity and readability make it an excellent choice for both beginners and experienced developers.
